Arduino Basic Input Output Circuits

Digital Out - 16x2 LCD Display

This example is a very straight forward example of interfacing an LCD display with the Arduinio. The Arduino IDE environment (Release 17 at time of writing) includes a number of pre-written code examples to use the LCD as well as having a pre-built library allowing very easy code writing for your own projects.

 

Again my specific hardware is the Arduino Duemilanove platform, but the concept is the same regardless of the exact platform.

 

Before continuing make sure you have read and understand the warning notice here.

 

Hardware Requirements;

1x Arduino

1x BC1602A LCD

1x 10kOhm Potentiometer

 

This circuit diagram for this example is shown below;

LCD Pin 1 is the ground

LCD Pin 2 is 5v supply

LCD Pin 3 is used to control the contrast of the display. This is achieved using the 10kOhm potentiometer.

LCD Pin 5 is Read/Write with Write being active LOW, this is tied to ground to allow writing of data to the LCD from the Arduino.

LCD Pins 15&16 are for the LCD Backlight.

The other pins used are control pins and data pins, these are connected to the Arduino and controlled by the Arduino.

 

You could easily extend this example by controlling the backlight and the contrast using the Arduino.

 

As stated at the top of the Arduino, a number of code examples are provided with the IDE, upload these to the Arduino to test you display.

Alternatively, upload the following code example to the Arduino;
